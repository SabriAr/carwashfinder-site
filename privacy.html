Politique de confidentialité

Dernière mise à jour : 14 mars 2026

1. Introduction

Carwash Finder ("l'Application") est éditée par Sabri A. Nous respectons votre vie privée et nous engageons à protéger vos données personnelles. Cette politique explique quelles données nous collectons, pourquoi et comment nous les utilisons.

2. Données collectées

L'Application collecte les données suivantes :

  • Identifiant d'appareil anonyme : un identifiant unique généré aléatoirement, non lié à votre identité. Il permet de gérer vos votes, favoris et contributions.
  • Position géographique : utilisée uniquement pour centrer la carte et afficher les stations de lavage à proximité. Votre position n'est jamais stockée sur nos serveurs.
  • Contributions volontaires : avis, notes, photos et suggestions que vous choisissez de partager.

3. Données NON collectées

  • Nom, prénom, email, numéro de téléphone
  • Contacts, calendrier, historique de navigation
  • Données de paiement

4. Utilisation des données

Les données collectées sont utilisées exclusivement pour :

  • Afficher les stations de lavage autour de vous
  • Permettre la contribution communautaire (votes, avis, suggestions)
  • Améliorer la qualité des informations affichées

5. Services tiers

L'Application utilise les services tiers suivants :

  • Supabase (base de données) : hébergé en Europe, conforme RGPD
  • OpenStreetMap / Overpass API : données cartographiques libres
  • Google AdMob : diffusion de publicités. AdMob peut collecter des identifiants publicitaires conformément à sa propre politique de confidentialité. Vous pouvez désactiver le suivi publicitaire dans les réglages de votre appareil.
  • Apple Maps / Google Maps : navigation vers les stations

6. Publicités

L'Application affiche des publicités via Google AdMob. Avant toute collecte d'identifiants publicitaires, l'Application vous demande votre consentement via la fenêtre App Tracking Transparency (ATT) d'Apple. Si vous refusez, seules des annonces non personnalisées sont affichées.

7. Stockage et sécurité

Vos données sont stockées de manière sécurisée sur les serveurs Supabase (chiffrement en transit et au repos). L'identifiant d'appareil est stocké localement dans le Secure Store de votre appareil.

8. Vos droits (RGPD)

Vous disposez des droits suivants :

  • Accès : vous pouvez consulter vos contributions dans l'application
  • Suppression : vous pouvez supprimer toutes vos données depuis Réglages > Supprimer mes données
  • Portabilité : vous pouvez exporter vos données depuis l'application

9. Conservation

Vos données sont conservées tant que vous utilisez l'application. Elles sont supprimées immédiatement et définitivement lorsque vous utilisez la fonction "Supprimer mes données".

10. Modifications

Cette politique peut être mise à jour. La date de dernière modification est indiquée en haut de cette page.
