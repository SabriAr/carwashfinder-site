Politique de confidentialité

Dernière mise à jour : 14 avril 2026

1. Introduction

Carwash Finder ("l'Application") est éditée par Sabri A. Nous respectons votre vie privée et nous engageons à protéger vos données personnelles. Cette politique explique quelles données nous collectons, pourquoi et comment nous les utilisons.

2. Données collectées

L'Application collecte les données suivantes :

  • Identifiant d'appareil stable : un identifiant unique généré localement et stocké dans le Secure Store de votre appareil. Il n'est pas lié à votre identité, mais il est envoyé avec chaque contribution (vote, favori, avis, signalement, ajout ou suggestion de carwash) et reste stable sur l'appareil pour rattacher vos contributions et permettre leur suppression.
  • Position géographique : utilisée pour centrer la carte et afficher les stations à proximité. Elle est également envoyée avec les carwashs que vous ajoutez à la communauté (coordonnées de la station).
  • Contributions volontaires : avis texte + note en étoiles, votes pour/contre, favoris, nouveaux carwashs ajoutés, suggestions de mise à jour, signalements. La version 1.0.1 n'inclut pas d'envoi de photos.

2 bis. Contenu public / contenu privé

Parmi les contributions :

  • Publiques dans l'app : les avis (texte + étoiles) sont visibles par les autres utilisateurs. Les nouveaux carwashs ajoutés par la communauté peuvent apparaître sur la carte après un dédoublonnage automatique côté serveur.
  • Non publiées : les suggestions de mise à jour et les signalements ne sont pas affichés dans l'app. Ils sont uniquement utilisés pour la modération.
  • Modération : chaque avis et chaque carwash peut être signalé via l'icône drapeau. Les contenus inappropriés sont supprimés manuellement après signalement, et les contributeurs abusifs peuvent être bloqués côté serveur.

3. Données NON collectées

  • Nom, prénom, email, numéro de téléphone
  • Contacts, calendrier, historique de navigation
  • Données de paiement

4. Utilisation des données

Les données collectées sont utilisées exclusivement pour :

  • Afficher les stations de lavage autour de vous
  • Permettre la contribution communautaire (votes, avis, suggestions)
  • Améliorer la qualité des informations affichées

5. Services tiers

L'Application utilise les services tiers suivants :

  • Supabase (base de données) : hébergé en Europe, conforme RGPD
  • OpenStreetMap / Overpass API : données cartographiques libres
  • Google AdMob : diffusion de publicités. AdMob peut collecter des identifiants publicitaires conformément à sa propre politique de confidentialité. Vous pouvez désactiver le suivi publicitaire dans les réglages de votre appareil.
  • Apple Maps / Google Maps / Waze : navigation vers les stations

6. Publicités

L'Application affiche des publicités via Google AdMob. Sur iOS, avant toute utilisation de l'identifiant publicitaire à des fins de suivi, l'Application vous demande votre consentement via la fenêtre App Tracking Transparency (ATT) d'Apple. Si vous refusez, seules des annonces non personnalisées sont affichées.

7. Stockage et sécurité

Vos données sont stockées de manière sécurisée sur les serveurs Supabase (chiffrement en transit et au repos). L'identifiant d'appareil est stocké localement dans le Secure Store de votre appareil.

8. Vos droits (RGPD)

Vous disposez des droits suivants :

  • Accès : vous pouvez consulter vos contributions dans l'application
  • Suppression : vous pouvez supprimer toutes vos données depuis Réglages > Supprimer mes données. Cette action supprime à la fois les lignes en base (avis, votes, favoris, contributions, signalements) et les fichiers éventuellement stockés dans notre stockage objet.
  • Export local : vous pouvez exporter depuis l'application une sauvegarde locale de certains réglages et préférences (par exemple favoris, thème et filtres).

9. Conservation

Vos données sont conservées tant que vous utilisez l'application. Elles sont supprimées immédiatement et définitivement lorsque vous utilisez la fonction "Supprimer mes données".

10. Modifications

Cette politique peut être mise à jour. La date de dernière modification est indiquée en haut de cette page.
